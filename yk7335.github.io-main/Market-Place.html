Covid-19 Themed Game Jam (Market Dash!)

Project information

  • Category: Game
  • Tech Used: C++, and Unreal Engine
  • Project date: April 2020
  • Project URL: https://yash22.itch.io/market-dash

During the start of the Covid-19 pandemic Texas A&M had a game jam. The goal was to create a game which highlighted the importance of the pandemic using information gathered by the CDC. My team and I developed a game which required the main character to follow social distancing guidlines while shopping around town for groceries. I was the project lead and we placed 9th overall.
