A web application that allows users to view recipes based on ingredients they have as well as related recipe names. My group and I utilized an API called Spoonacular to obtain ingredient and recipe information. It also displays nutritional information about each recipe selected by the user. It allows users to view their recommended calories using a BMI API and takes into account any excerice they do using the Nutritionix API. Some accessibility features our web application supports is that it allows the users to print out the recipes in a pdf format and a text to speech where it plays the steps to each recipies.
